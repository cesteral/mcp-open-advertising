# AGENTS.md

Guidance for AI coding agents working in this repository lives in
[CLAUDE.md](CLAUDE.md). That file covers architecture patterns,
conventions, auth-mode rules, and the cross-cutting subsystems
(session services, tool-failure logging, report-CSV spill).

The public fleet overview — server list, ports, API versions, and
tool counts — is in [README.md](README.md).

This file exists so agents following the AGENTS.md convention
(e.g. non-Claude tooling) can find the guidance without guessing.
